    # Vue (Part I)

    # Description

    The goal of this assignment is to get familiar with Vue.js.

    Choose one of the following topics and create your first Vue application.

    • Calculator
    • To-Do
    • Weather
    • Alarm Clock
    • Wallet

    # Projects

    # Calculator

    Implement a calculator with following characteristics.

    • Basic calculation
      • Addition
      • Subtraction
      • Multiplication
      • Division
    • Conversion
      • Decimal - Binary - Octal - Hexadecimal
    • History

    # To-Do

    Implement a To-Do application with following characteristics.

    • Creation
    • Deletion
    • Information
      • Author
      • Date

    # Weather

    Implement a weather forecast application with following characteristics.

    • Input
      • Location
    • Output
      • Temperature
      • Cloudiness
      • Humidity
      • Geo coordination
    • Forecast
      • 1-Day (detail)
      • 3-Day
      • 1-Week

    # Alarm Clock

    Implement an alarm clock with following characteristics.

    • Creation
    • Deletion
    • Sound Selection

    # Wallet

    Implement a wallet application with following characteristics.

    • Creation
    • Deletion
    • History
    • Statistics
    • Overview (Category)
