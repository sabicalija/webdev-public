# Assignment 1: HTML, CSS, JavaScript

# Description

Create a (simple) web page where users can create and collect notes. During this assignment, you should learn or repeat the basics of HTML, CSS, and JavaScript. You can use following pen (CodePen) as a starting point.

Note

Keep in mind, that the web page (HTML) - other than the CodePen online examples - needs <link> elements to the style sheet (CSS) and script (JS) if you assign them individually.

# Tips

  • Use the HTML elements with care and lookup the semantics of an element before considering using it.

# Requirements

  • Users can create new notes, where they have to enter
    • a note
    • a date
    • the name of a person
  • Users can read/lookup following information of created notes
    • the note
    • the date
    • the name of a person
    • the date of creation (note)
  • Users can delete created notes

# Extension

After implementing the functionality from above, extend the app.

  • Users can create new lists, where they have to enter
    • a name
  • Users can create new notes as previously, but the notes have now to be added to existing lists.

# Further ideas

Following tasks are purely optional and have not to be implemented for the next course lecture. However, if you are done with the tasks from above, you can use the following list of features as an inspiration to add further functionality or implement some ideas on your own.

  • Search
    Implement a search bar, allowing users to search through created notes.
  • Storage
    Every page reload or restarting the browser leads to a complete restart of the app and all notes are lost. Use the Web Storage API (Tutorial, Reference) (or other means) to persist created notes.

# Starting Point

index.html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Alija Sabic: ToDo App</title>
    <link rel="stylesheet" href="./style.css">
  </head>
  <body>
   <ul id="list">
     <li>Item 1</li>
     <li>Item 2</li>
     <li>Item 3</li>
     <li>Item 4</li>
     <li>Item 5</li>
   </ul>
    <form>
      <input id="text" type="text">
      <input id="button" type="submit" value="Add Note">
    </form>
    <script  src="./script.js"></script>
  </body>
</html>

style.css

* {
  outline: 1px black dashed;
  padding: 3px;
  margin: 3px;
}

body {
  outline: 2px red solid;
}

ul {
  list-style-type: none;
}

script.js

// -------------------------------------
// DOM API:
// -------------------------------------
// Methods
// + getElementById(id: string)
// + createElement(elementName: string) 
// + appendChild(element: object)
// -------------------------------------
// Properties
// + innerText

// 1. Get a reference to the list
var list = document.getElementById("list");

// 2. Create a new `li` element
var item = document.createElement("li");
  
// 3. Write text to new `li` element
item.innerText = "Item 6";

// 4. Append new `li` element to the list
list.appendChild(item);
