        # HTML, CSS, JavaScript

        # Description

        Create a (simple) web application where users can complete a quiz. During this assignment, you should learn or repeat the basics of HTML, CSS, and JavaScript.

        Max. 20 pts

        # Tips

        • Use the HTML elements with care and lookup the semantics of an element before considering using it.

        # Requirements

        • Register an user account at QuizAPI
        • Create a web application that lets user answer questions (requested from QuizAPI, 7 pts)
          • Create a GitHub repository of your project (GitHub)
          • Implement basic functionality where users can answer questions upon page reloads (Fetch API, XMLHttpRequest)
        • Let the user request new questions without page reloads (3 pt)
        • Let the user configure category, difficulty, tags, etc. for each question (4 pt)
        • Let the user complete a round-based quiz game (5 pt)
        • Let the user save and display high-scores (5 pt, Storage API)
        • Create a responsive page design (5 pt, Mobile, Desktop, Flexbox)
        • Publish game via GitHub pages (5 pts, GitHub Pages)
